# Companies Directory - Frontend

React + TypeScript frontend application for the Companies Directory.

## Setup

1. Install dependencies:
```bash
npm install
```

2. Create `.env` file (optional, defaults to proxy):
```env
VITE_API_URL=http://localhost:3001/api
```

3. Start the development server:
```bash
npm run dev
```

The application will be available at `http://localhost:3000`

## Build

```bash
npm run build
```

The build output will be in the `dist` directory.

## Preview Production Build

```bash
npm run preview
```

## Features

- Company listing with grid layout
- Real-time search and filtering
- Sorting by name or founded year
- Pagination
- Loading, error, and empty states
- Responsive design
- Accessibility features
